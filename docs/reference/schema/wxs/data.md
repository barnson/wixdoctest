# Data element
Used for a Custom Table. Specifies the data for the parent Row and specified Column.

## Parents
[Row](row.md)

## Attributes
**Column** (String, required)
  : Specifies in which column to insert this data.

**Value** (String)
  : Specifies the optional data for this column.
