# Tab element (Thmutil extension)
Defines an individual tab within a set of tabs.

## Parents
[Tabs](tabs.md)

## Attributes
**StringId** (xs:nonNegativeInteger)
  : Identifier that references a string resource in the module to define the caption of the tab.
